# Прототип RAG-системы для автоматизации поиска терапевтических мишеней болезни Альцгеймера. Создано в рамках тестового задания BIOCAD 2026.

## Функционал
- Сбор статей из PubMed по заданным ключевым словам.
- Использование LangChain и OpenAI GPT-4o для генерации ответов.
- Векторное хранилище FAISS для быстрого поиска по контексту.
- Интерактивный интерфейс на Streamlit.

## Как запустить
1. Клонируйте репозиторий.
2. Установите зависимости: `pip install -r requirements.txt`.
3. Создайте файл `.env` и добавьте `OPENAI_API_KEY=ваш_ключ`.
4. Запустите приложение: `streamlit run app.py`.
